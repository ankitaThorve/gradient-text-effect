# 🌈 Gradient Text Effect

A simple and modern gradient text effect built using HTML and CSS.
This project demostrates how to create visualling appealing text using background gradients and clipping techniques.

## ⚙️ How It Works

The effect is created using:

- `background: linear-gradient(...)`
- `-webkit-background-clip: text`
- `-webkit-text-fill-color: transparent`

This allows the gradient to be applied inside the text instead of the background.

## 📸 Customization

You can easily customize the gradient colors:

```css
background: linear-gradient(to right, #6a11cb, #ff0080);
```
Try different directions and colors to create unique effects.
